```

📌 Status: Active & Maintained
⭐ Stars: 1+
```

**What It Does:**
- Automated data extraction from OpenWeather API & climate datasets
- Multi-stage transformation pipeline (Jayvee + Python)
- Data validation & quality checks
- SQLite database management

- Statistical analysis & visualization

**Tech Stack**: Python, Jayvee, SQL, GitHub Actions, Pandas

**Pipeline Flow:**
```
Raw API Data 
    ↓ [Jayvee Extraction & Validation]

    ↓ [Analysis & Insights]
Visualizations & Reports
```

**Key Achievements:**
- ✅ Processes 99.2% complete data after cleaning
- ✅ Fully automated (runs daily via GitHub Actions)
- ✅ Modern data engineering practices (Jayvee, Python)
- ✅ Real-world climate data (CO₂, rainfall)
- ✅ Industry-standard ETL patterns

**Why It Matters**: Demonstrates professional data engineering skills
